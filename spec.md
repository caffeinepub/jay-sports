# Specification

## Summary
**Goal:** Update the HTML metadata in `frontend/index.html` to reference the domain `jaysports.in`.

**Planned changes:**
- Update the `<title>` tag to `jaysports.in` or `Jay Sports | jaysports.in`
- Update the canonical `<link>` href to reference `jaysports.in`
- Update Open Graph tags (`og:url`, `og:site_name`) to reference `jaysports.in`
- Update Twitter Card meta tags to reference `jaysports.in`
- Remove any old placeholder or staging domain references from `index.html`

**User-visible outcome:** The browser tab and all HTML metadata consistently display and reference `jaysports.in` as the site's domain.
